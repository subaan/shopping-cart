Home slider

Home slider use owlcarousel, For more details see plugin's page: http://www.owlcarousel.owlgraphic.com/

First, we need to include owlcarousel css & js file

<link href="css/owl.carousel.css" rel="stylesheet">
<link href="css/owl.theme.default.css" rel="stylesheet">

<!-- Also Include this js just before custom script section -->

Customize Style

This template now use SASS CSS pre-processor, all style stored in style.scss

You can customize bacis variable in style.scss

<!-- Bootstrap mixin required for generate button and styling some vendor prefix -->
@import '../bootstrap-sass/bootstrap/mixins';
<!-- Some helper from bootstrap 4 -->
@import 'helper';
<!-- Robotodraft Font Family -->
@import 'robotodraft';

$font-family-base: 'RobotoDraft','Roboto',sans-serif;
$font-color-base: #666;
$border-radius-base: 0;

$font-color-footer: #bababa;
$background-color-footer: #373939;

There is variable $border-radius-base, to make flat style set the value to 0
If you want to make rounded style set $border-radius-base value to higher than 0 (4px for example on this template)

You also can customize color style. The color style stored in _color.scss
All color style used by this template is based from material design color https://material.google.com/style/color.html#color-color-palette 

// Teal
$color-50 : #e0f2f1;
$color-200 : #80cbc4;
$color-400 : #26a69a;
$color-500 : #337ab7;
$color-600 : #00897b;
$color-700 : #00796b;

Compile New Style

After customizing your style, you need to compile scss file to css file.
Before that, make sure you have installed sass. For more details see http://sass-lang.com/install

Compile your style using this command

sass -t expanded style.scss:new_style.css

For compile minified version use this command

sass -t compressed style.scss:new_style.min.css
